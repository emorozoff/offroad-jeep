# Offroad Jeep — браузерная 3D-песочница на внедорожнике

Свободная езда на внедорожнике от третьего лица. Физика — ядро геймплея: реалистичная,
но играбельная. Никаких таймеров и гонок — катаешься в удовольствие, исследуешь карты,
прыгаешь с трамплинов, буксуешь, переворачиваешься.

Вся игра — в одном файле **`index.html`**. Сборка и сервер не нужны.

## Запуск

Просто откройте `index.html` в современном браузере (Chrome / Edge / Firefox / Safari).
Нужен интернет — Three.js и cannon-es подгружаются с CDN (jsdelivr) через importmap.

В меню выберите карту и машину и нажмите **«Поехали»**.

## Управление

| Клавиши | Действие |
|---|---|
| **W / ↑** | газ |
| **S / ↓** | тормоз / задний ход |
| **A · D / ← ·→** | руль |
| **Пробел** | ручник |
| **R** | перевернуть / вернуть машину |
| **Esc** | пауза / выход в меню |
| **M** | звук вкл/выкл |
| **Мышь** (зажать и вести) | осмотреться вокруг машины |
| **Колесо мыши** | зум камеры |

HUD: спидометр (км/ч), текущая карта · машина · поверхность, подсказки по клавишам.

## Карты

Каждая карта — процедурный террейн (шум + FBM), своё небо, освещение и туман,
и своё поведение поверхности:

- **Пустыня** — песчаные дюны и каньоны. Песок мягкий: машина скользит и теряет
  сцепление на крутых дюнах.
- **Зима** — снежные поля и замёрзшее озеро. Лёд — очень низкое сцепление (заносы),
  снег — вязкий, тормозит.
- **Джунгли** — сплошная грязь. Высокое сопротивление: машина буксует и скользит.
  Густой туман — ограниченная видимость.

На каждой карте расставлены препятствия: камни, трамплины (с них можно прыгать) и брёвна.

## Машины

Три машины с разными характеристиками (масса, мощность, сцепление, подвеска, привод):

- **Багги** — лёгкий и быстрый, задний привод, верткий, охотно срывается в занос.
- **Джип** — сбалансированный вездеход, полный привод, предсказуемый.
- **Внедорожник** — тяжёлый и мощный, большие колёса и длинноходная подвеска,
  медленный, но устойчивый.

## Технологии

- **Three.js** r184 — рендеринг (через importmap, jsdelivr).
- **cannon-es** 0.20.0 — физика; машина на `RaycastVehicle`.
- Террейн — `CANNON.Heightfield`, совмещённый с визуальным мешем из той же карты высот.
- Звук — процедурный WebAudio (двигатель, визг шин, удар при приземлении), без внешних файлов.
- Графика — стилизованные примитивы и процедурная геометрия. `GLTFLoader` подключён
  через importmap — модели CC0 (Kenney/Quaternius) можно подменить отдельным шагом.

Весь код — в `index.html`. План разработки и журнал — в `plan.md` и `progress.md`.
