# Прогресс

Отметки о выполненных майлстоунах и принятых решениях. См. `plan.md`.

## Технический стек (зафиксировано)
- Three.js **r184** (`0.184.0`), addons из `examples/jsm/` — через importmap (jsdelivr).
- cannon-es **0.20.0** (`dist/cannon-es.js`) — ESM с jsdelivr.
- Машина: `CANNON.RaycastVehicle` (4 колеса-рейкаста). Мир: `SAPBroadphase`, `world.step(1/60, dt, 3)`.
- Верификация: локальный статик-сервер + Playwright (navigate → ждать → console_messages → screenshot). Цель: ноль ошибок в консоли.
- Git-репозиторий создан отдельно в папке проекта (домашняя папка — тоже репо, в неё НЕ коммитим).

- [x] **M6 — Система поверхностей и 3 карты** ✓ (2026-06-19)
  - Рефактор в `loadMap(map)`: чистый снос (dispose геометрий/материалов, removeBody террейна/стен/препятствий) и пересборка террейна+препятствий+окружения; счётчик тел в мире не растёт между картами.
  - 3 карты `MAPS`: **Пустыня** (большие дюны, песчаник, тёплый свет), **Зима** (снежные поля + замёрзшее озеро через `lakeLevel`-клэмп, лёд/снег зонально, холодный синий), **Джунгли** (мшистые холмы, плотный зелёный туман/мрак). У каждой свои terrain/env(небо,туман,свет)/obstacles.

  - HUD: бейдж «карта · поверхность» обновляется на лету.
  - Верификация (числа): переключение всех карт без ошибок и без утечки тел; coast-decel лёд 4.6 < грунт 5.7 < песок 7.1 < снег 9.0 < грязь 9.4 м/с²; поворот на 1.3с: грунт 77° vs лёд 20° (лёд тащит прямо — занос/недостаток сцепления). Скриншоты 3 карт — явно разные. 0 ошибок.
  - Тест-хук `DBG.setSurface(s)` для замера поведения на любой поверхности.
