# Прогресс

Отметки о выполненных майлстоунах и принятых решениях. См. `plan.md`.

## Технический стек (зафиксировано)
- Three.js **r184** (`0.184.0`), addons из `examples/jsm/` — через importmap (jsdelivr).
- cannon-es **0.20.0** (`dist/cannon-es.js`) — ESM с jsdelivr.
- Машина: `CANNON.RaycastVehicle` (4 колеса-рейкаста). Мир: `SAPBroadphase`, `world.step(1/60, dt, 3)`.
- Верификация: локальный статик-сервер + Playwright (navigate → ждать → console_messages → screenshot). Цель: ноль ошибок в консоли.
- Git-репозиторий создан отдельно в папке проекта (домашняя папка — тоже репо, в неё НЕ коммитим).
