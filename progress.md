# Прогресс

Отметки о выполненных майлстоунах и принятых решениях. См. `plan.md`.

## Технический стек (зафиксировано)
- Three.js **r184** (`0.184.0`), addons из `examples/jsm/` — через importmap (jsdelivr).
- cannon-es **0.20.0** (`dist/cannon-es.js`) — ESM с jsdelivr.
- Машина: `CANNON.RaycastVehicle` (4 колеса-рейкаста). Мир: `SAPBroadphase`, `world.step(1/60, dt, 3)`.
- Верификация: локальный статик-сервер + Playwright (navigate → ждать → console_messages → screenshot). Цель: ноль ошибок в консоли.
- Git-репозиторий создан отдельно в папке проекта (домашняя папка — тоже репо, в неё НЕ коммитим).

## Майлстоуны
- [x] **M1 — Скелет проекта и рендер-сцена** ✓ (2026-06-19)
  - importmap (three r184 + cannon-es 0.20.0) грузится без ошибок; рендерер с тенями (PCFShadowMap), ACES tonemapping.
  - Градиентное небо (sky dome shader), туман, hemisphere+directional свет, плоский пол, тени.
  - Цикл рендера на `performance.now()` delta (без deprecated Clock/Timer), ресайз обновляет камеру и буфер.
  - Оверлей ошибок (`#fatal`) + `window.__errors` для headless-проверки; `window.GAME` для дебага.
  - Верификация Playwright: 0 ошибок/0 предупреждений, ~35fps, скриншот сцены с тенями, ресайз 900×600 → aspect 1.5 ✓.
  - Решение: WebGL в headless Chrome требует `--use-angle=swiftshader --enable-unsafe-swiftshader`.
- [x] **M2 — Физика и управляемая машина на плоскости** ✓ (2026-06-19)
  - `CANNON.World` (SAPBroadphase, gravity, `world.step(1/60, dt, 3)`), статический пол-плоскость, выровнен с визуалом.
  - Конфиг-объект машины + фабрика `createVehicle(cfg)`: chassis Box + `RaycastVehicle` с 4 колёсами; визуал — кузов+кабина+нос-маркер (+z=вперёд) и колёса-цилиндры со спицами.
  - Управление WASD/стрелки: газ (-engineForce=вперёд), тормоз, реверс из стопа, плавный руль (front wheels), Пробел — ручник (блок задних).
  - Синхронизация мешей через `updateWheelTransform`/`worldTransform`.
  - **Решение (важно):** низкий центр масс через offset коллизийного бокса и визуала вверх от physics-origin (`comOffsetY`). Снизило «вилли» при разгоне с ~19° до **2.1°**, высота посадки не изменилась.
  - Верификация Playwright: 0 ошибок; вперёд +18м/1.5с (~64км/ч), руль (yaw +33°), тормоз 15.9→3.5, реверс −21.5, ручник 15.9→7.4; скриншоты ОК.
