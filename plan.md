# План: браузерная 3D-игра «Оффроад-джип» (песочница)

Свободная езда на внедорожнике, вид от третьего лица, реалистичная-но-играбельная физика.
Один файл `index.html`, без сборки и без сервера. Three.js (r184) + cannon-es (0.20.0) с CDN
через importmap (jsdelivr). Машина — `CANNON.RaycastVehicle`.

Идём строго по майлстоунам, по одному за раз: реализовать → проверить (headless через Playwright,
ноль ошибок в консоли) → `git commit` → отметить в `progress.md` → дальше.

## Ядро (core)

### M1 — Скелет проекта и рендер-сцена
- `index.html` с importmap (three + addons + cannon-es), canvas на весь экран.
- Three.js сцена: камера, рендерер, освещение (hemisphere + directional + тени), плоский «пол», небо/фон, туман.
- Цикл рендера (requestAnimationFrame), ресайз окна.
- **Готово:** страница открывается, в консоли ноль ошибок, видна 3D-сцена с полом и тенями, окно ресайзится.

### M2 — Физика и управляемая машина на плоскости
- `CANNON.World` (гравитация, SAPBroadphase, фиксированный шаг `world.step(1/60, dt, 3)`).
- Статический физический пол + визуальный пол совмещены.
- Машина-болванка: chassis (Box) + 4 колеса через `RaycastVehicle`/`addWheel`.
- Колёса-цилиндры визуально, синхронизация `updateWheelTransform` → меши.
- Управление WASD/стрелки: газ/тормоз/задний ход, руль; Пробел — ручник.
- **Готово:** болванка-джип ездит по плоскости, рулит, тормозит, едет назад; колёса крутятся и поворачиваются; ноль ошибок.

### M3 — Камера от третьего лица, мышь, зум, HUD
- Камера следует за машиной (плавно, с учётом скорости/направления).
- Мышь — осмотреться вокруг машины (orbit), колесо — зум, авто-возврат за машину при езде.
- R — перевернуть/вернуть машину на колёса.
- HUD: спидометр (стрелочный + цифры км/ч), подсказки по клавишам.
- **Готово:** камера красиво следует, мышь вращает обзор, зум работает, R поднимает машину, HUD показывает скорость.

### M4 — Процедурный террейн (heightmap из шума)
- Шумовая функция (value/FBM noise, детерминированная по seed).
- `CANNON.Heightfield` + совмещённый `PlaneGeometry`, выверенная ориентация/индексация.
- Машина едет по холмам, физика совпадает с визуалом.
- **Готово:** холмистый террейн, машина уверенно ездит по нему, коллизия совпадает с рельефом, ноль ошибок.

### M5 — Препятствия и сброс
- Камни (сферы/выпуклые), трамплины (наклонные боксы), кочки/неровности — физика + меши.
- Трамплины реально подбрасывают машину; камни сталкиваются.
- R-сброс надёжно работает и на рельефе (поднять, выровнять, поставить выше поверхности).
- **Готово:** препятствия видимы и сталкиваются, трамплин подбрасывает, R-сброс работает везде.

## Расширение (content)

### M6 — Система поверхностей и 3 карты
- Три карты со своим рельефом, небом, освещением, туманом:
  1. **Пустыня** — дюны/каньоны, песок: мягкий, лёгкое проваливание/скольжение, потеря сцепления на крутых дюнах.
  2. **Зима** — замёрзшее озеро (лёд, очень низкое сцепление, заносы) + снежные поля/сугробы (вязко, тормозит).
  3. **Джунгли** — грязь: высокое сопротивление, буксует и скользит.
- Параметры поверхности (frictionSlip, сопротивление/вязкость, цвет, частицы) задаются per-map (и зонально: лёд vs снег).
- **Готово:** 3 карты, у каждой явно различимое поведение поверхности, своё небо/туман/свет; переключение между ними работает.

### M7 — Несколько машин (≥3)
- Минимум 3 машины с разными характеристиками: масса, мощность, сцепление, жёсткость подвески, привод (FWD/RWD/AWD).
  Напр.: лёгкий быстрый багги, средний джип, тяжёлый внедорожник.
- Конфиг-объекты, фабрика собирает `RaycastVehicle` по конфигу; визуально различимы (размер/цвет/форма болванок).
- **Готово:** ≥3 машины, выбираются, ощущаются по-разному (разгон, занос, крены, проходимость).

### M8 — Главное меню и навигация
- Меню при запуске: выбор карты, выбор машины, кнопка «Старт».
- Из игры — пауза/выход в меню (Esc), возврат и рестарт с другим выбором.
- **Готово:** полный цикл меню→игра→меню работает, выбор карты и машины применяется.

## Полиш (polish)

### M9 — Частицы, следы, тряска камеры
- Частицы из-под колёс по поверхности: пыль (пустыня), снежная крошка (зима), брызги грязи (джунгли) — только при контакте/пробуксовке.
- Следы шин на поверхности (decals/затухающие линии).
- Тряска камеры на кочках/жёстких приземлениях; крен на скорости.
- **Готово:** частицы корректные по карте и появляются только при движении/буксе; следы остаются; камера трясётся на кочках.

### M10 — Звук, ассеты, баланс, производительность
- Процедурный звук через WebAudio (двигатель по оборотам, визг шин при заносе, удар на приземлении) — без внешних ассетов.
- Опционально: подмена болванок на CC0 low-poly модели (Kenney/Quaternius) через GLTFLoader с надёжным фоллбэком на примитивы; HDRI-небо (Poly Haven) при наличии.
- Балансировка ощущений (золотая середина аркада/симулятор), тюнинг per-surface, чистка консоли, целевой FPS.
- **Готово:** есть звук двигателя/шин, всё стабильно (нет падения FPS), опциональные ассеты грузятся с фоллбэком, игра ощущается отзывчиво и реалистично.

### M11 — Финальная QA и сведение
- Полный прогон всех карт × всех машин: загрузка, езда, прыжки, перевороты, R-сброс, меню.
- Проверка отсутствия ошибок/утечек в консоли, корректность на ресайзе.
- README с управлением и описанием.
- **Готово:** все майлстоуны выполнены, игра проходит сквозной прогон без ошибок, README на месте.
